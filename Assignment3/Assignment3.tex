\section{Introduction}
In the previous assignment, I evaluated the following branch predictors:
\begin{itemsize}
\item Load Value Correlator Predictor (LVCP)
\item Register-Value-Aware Branch Predictor (RVA-Toru)
\item TAGE-SC 2025 André Seznec Predictor
\item TAGE-SC-L Alberto Ros Predictor
\end{itemsize}
In this report, I will continue my analysis of these predictors by performing a similiar analysis of the remaining CBP2025 entries, namely
\begin{itemsize}
        \item MultiPerspective Perceptron Predictor (MPP) by Jimenez, et al.
        \item Programming Idiom Preictor (PIP)
        \item Branch Prediction vua Load Value Prediction ( BALL ) by Jun Fan
        \item The Bulleseye Predictor by Emet Behrendt, et al.
\end{itemsize}
I will do a similiar analysis of these predictors as I did for the first four and then I will compare all 8 predictors together to draw a final conclusion
and rank them based on their performance.
